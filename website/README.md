# Сайт Boneka Client

Дизайн в тёмной теме с фиолетовыми акцентами (как на референсах 11–14).

## Что уже есть
- **Главная** (`index.html`) — лендинг с кнопкой регистрации.
- **Регистрация** (`register.html`) — форма: почта, логин, пароль, соглашение. **hCaptcha подключён** (скрипт `https://js.hcaptcha.com/1/api.js`). В форме стоит блок `<div class="h-captcha" data-sitekey="YOUR_SITEKEY"></div>` — замените `YOUR_SITEKEY` на ключ с [dashboard.hcaptcha.com](https://dashboard.hcaptcha.com).
- **Стили** (`style.css`) — мыльное стекло / тёмная тема.
- **Скрипт** (`app.js`) — заглушка под API.

## Что нужно добавить

1. **Бэкенд** (любой язык: Node.js, PHP, Python, Go):
   - Сохранение пользователей (БД).
   - API: регистрация, вход, профиль, настройка способа оплаты.
   - Привязка лицензии к HWID (HWID приходит с лаунчера/клиента, сохраняете в профиле).

2. **Проверка на бота (CAPTCHA)**  
   Подключите на странице регистрации:
   - [hCaptcha](https://www.hcaptcha.com/) или  
   - [Google reCAPTCHA v3](https://developers.google.com/recaptcha/docs/v3)  
   Отправляйте токен на бэкенд и проверяйте его там.

3. **Анти-DDoS**  
   - Поставьте сайт за [Cloudflare](https://www.cloudflare.com/) (или аналог).  
   - Включите защиту от DDoS в панели.

4. **Способ оплаты и профиль**  
   - В личном кабинете (страница после входа): выбор способа оплаты, привязка HWID, история.  
   - Реализуется через бэкенд + отдельные страницы/формы.

5. **HWID**  
   - Лаунчер уже считает HWID и при первом запуске привязывает устройство.  
   - При покупке/активации можно отправлять HWID на сайт и сохранять в профиле (1 лицензия = 1 HWID или по вашим правилам).

## Запуск и бесплатный домен/ссылка
- **Локально:** откройте `index.html` в браузере или выполните в папке `website`: `npx serve .` — сайт будет по адресу типа `http://localhost:3000`.
- **Бесплатный хостинг (получить ссылку на сайт):**
  - **[GitHub Pages](https://pages.github.com/):** создайте репозиторий, залейте папку `website` в ветку `main` в папку `docs` или включите GitHub Pages для корня; ссылка будет `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО`.
  - **[Netlify](https://www.netlify.com/):** зарегистрируйтесь, перетащите папку `website` в Drop Deploy — Netlify выдаст бесплатный URL вида `https://случайное-имя.netlify.app` (можно задать своё поддомен).
  - **[Vercel](https://vercel.com):** аналогично, импорт проекта или загрузка папки — бесплатный домен.
- Для регистрации и оплаты нужен бэкенд (API); CAPTCHA проверяется на сервере через [hCaptcha siteverify](https://docs.hcaptcha.com/#verifying-the-user-response-server-side).

Папка загрузок
==============

Чтобы на сайте работали кнопки скачивания:

1. boneka-launcher.jar — уже лежит в папке launcher/ (копия не нужна).

2. Исходный код чита (архив):
   Положите сюда файл boneka-source.zip с исходным кодом клиента.
   Тогда кнопка «Скачать исходный код» будет вести на этот файл.

3. Лаунчер + исходник одним архивом:
   Создайте boneka-full.zip, в который положите:
   - boneka-launcher.jar (из папки launcher/)
   - папку или архив с исходным кодом (например, expensive-master или boneka-source.zip)
   Положите boneka-full.zip в эту папку (downloads/).
   Тогда кнопка «Скачать лаунчер и исходник» будет вести на этот файл.

Имена файлов на сайте:
  downloads/boneka-source.zip  — только исходник
  downloads/boneka-full.zip    — лаунчер + исходник
